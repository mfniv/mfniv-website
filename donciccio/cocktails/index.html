Classics — Bartender Build Reference

Pre-Batched Classics

Negroni
🍸 Rocks | 🍋 Lemon Peel | Vessel: In Glass
  1. 🧊 Add large rock ice (1 cube)
  2. 🥃 Pour Negroni Batch (2.5–3.0 oz)
  3. 🥄 Brief stir
  4. 💨 Express lemon peel (1)
  5. 🍋 Place or discard peel
Manhattan
🍸 Coupe | 🍊 Orange Peel | Vessel: Mixing Glass
  1. 🥃 Add Manhattan Batch (2.5–3.0 oz)
  2. 🧊 Add ice (if stirring)
  3. 🥄 Stir briefly
  4. 🔽 Strain into coupe
  5. 💨 Express orange peel (1)
  6. 🍊 Finish garnish
Old Fashioned
🍸 Rocks | 🍊 Orange Peel | Vessel: In Glass
  1. 🧊 Add large rock ice (1 cube)
  2. 🥃 Pour Old Fashioned Batch (2.25–2.75 oz)
  3. 🥄 Brief stir
  4. 💨 Express orange peel (1)
  5. 🍊 Place peel in glass

Non-Batched Classics

Americano
🍸 Tall | 🍊 Orange Wheel | Vessel: In Glass
  1. 🥃 Add Cinque (1.0 oz)
  2. 🥃 Add Vermouth Bianco (1.0 oz)
  3. 🧊 Add ice (fill)
  4. ⬆️ Top with soda
  5. 🍊 Add orange wheel (1)
Boulevardier
🍸 Coupe | 🍊 Orange Peel | Vessel: Mixing Glass
  1. 🥃 Add Bourbon (0.75 oz)
  2. 🥃 Add Donnarosa (0.75 oz)
  3. 🥃 Add Sweet Vermouth (1.0 oz)
  4. 🧊 Add ice (fill)
  5. 🥄 Stir
  6. 🔽 Strain into coupe
  7. 💨 Express orange peel (1)
Spritz
🍸 Wine Glass | 🍊 Orange Wheel | Vessel: In Glass
  1. 🥃 Add Ambrosia (1.5 oz)
  2. 🧊 Add ice (fill)
  3. 🥃 Add Prosecco (2.0 oz)
  4. ⬆️ Top with soda (splash)
  5. 🍊 Add orange wheel (1)
